# 大象学长官网 · 操作手册

## 一、初次部署（只需做一次）

### 1. 安装工具

打开 Mac 的 **终端**（Terminal），依次运行：

```bash
# 安装 Node.js（如果没装过）
brew install node

# 安装 Git（如果没装过）
brew install git

# 验证安装
node -v
git -v
```

如果没有 Homebrew，先安装它：
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### 2. 创建 GitHub 仓库

1. 打开 https://github.com ，注册/登录
2. 点右上角 "+" → "New repository"
3. 名字填 `daxiangxuezhang.com`，选 Private，点 Create

### 3. 上传项目到 GitHub

在终端运行（把 `你的用户名` 换成你的 GitHub 用户名）：

```bash
cd ~/Desktop/daxiangxuezhang.com
git init
git add .
git commit -m "初始版本"
git remote add origin https://github.com/你的用户名/daxiangxuezhang.com.git
git branch -M main
git push -u origin main
```

### 4. 部署到 Vercel

1. 打开 https://vercel.com ，用 GitHub 账号登录
2. 点 "Add New Project"
3. 选择 `daxiangxuezhang.com` 仓库，点 Import
4. 直接点 Deploy，等 1-2 分钟
5. 部署成功后会给你一个 `.vercel.app` 的临时域名

### 5. 绑定你的域名

在 **Vercel** 项目设置中：
1. Settings → Domains → 输入 `www.daxiangxuezhang.com` → Add

在 **Cloudflare** DNS 设置中：
1. 添加一条 CNAME 记录：
   - Name: `www`
   - Target: `cname.vercel-dns.com`
   - Proxy status: 关掉代理（灰色云朵）
2. 添加另一条 CNAME 记录（裸域名）：
   - Name: `@`
   - Target: `cname.vercel-dns.com`
   - Proxy status: 关掉代理

等几分钟生效后，访问 www.daxiangxuezhang.com 就能看到你的网站了。

## 二、发新视频（日常操作）

每次 B站发了新视频，只需 3 步就能同步到网站：

### 步骤 1：创建内容文件

在 `content/videos/` 文件夹里，复制一个已有的 `.md` 文件，改个文件名，然后编辑里面的内容。

文件名规则：用英文短横线，比如 `sora-movie-workflow.md`

文件格式：

```markdown
---
title: "视频标题"
bvid: "BV1xxxxxxxxx"
category: "工作流"
date: "2026-04-15"
duration: "18:24"
views: "12.3万"
isNew: true
tags: ["Sora", "工作流"]
description: "一句话描述这期视频讲什么"
resources:
  - name: "附件显示名.pdf"
    file: "actual-filename.pdf"
    size: "2.1 MB"
relatedCourse:
  title: "对应课程名"
  description: "推荐理由"
  price: "¥2,799"
---

这里写配套笔记的正文...

## 第一部分标题

笔记内容...

## 第二部分标题

笔记内容...
```

**bvid 在哪找？** 打开B站视频页面，网址里的 `BV` 开头的那串字符就是。
比如 `https://www.bilibili.com/video/BV1xx411c7mD/`，bvid 就是 `BV1xx411c7mD`

### 步骤 2：放入附件（如果有）

把 PDF、模板等附件文件放到 `public/resources/` 文件夹里。
文件名要和 `.md` 里写的 `file` 字段对应。

### 步骤 3：推送上线

打开终端，运行：

```bash
cd ~/Desktop/daxiangxuezhang.com
git add .
git commit -m "新视频：这里写视频标题"
git push
```

完成！大约 30 秒后网站自动更新。

## 三、项目文件结构说明

```
daxiangxuezhang.com/
├── content/
│   └── videos/          ← 你的视频内容都放这里（.md 文件）
├── public/
│   └── resources/       ← 附件资源放这里（PDF、模板等）
├── src/
│   ├── app/             ← 页面代码（一般不用动）
│   ├── components/      ← 组件代码（一般不用动）
│   └── lib/             ← 工具代码（一般不用动）
├── package.json
└── 操作手册.md          ← 就是这个文件
```

**你日常只需要关心两个文件夹：**
- `content/videos/` — 写视频笔记
- `public/resources/` — 放附件

其他文件夹里的代码不需要动。

## 四、常见问题

**Q: 想取消某个视频的 NEW 标记？**
把 `.md` 文件里的 `isNew: true` 改成 `isNew: false`，然后推送。

**Q: 想修改已发布视频的笔记？**
直接编辑对应的 `.md` 文件，然后推送。

**Q: 忘记 git 命令了？**
```bash
git add .
git commit -m "更新内容"
git push
```
就这三行，每次都一样。

**Q: 推送报错了？**
多半是网络问题，再试一次。如果还不行，截图问我。
